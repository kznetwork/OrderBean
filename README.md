# OrderBean 🫘

OrderBean은 주문 관리 시스템을 위한 Toy Project입니다.

## 📋 프로젝트 개요

OrderBean은 간단하고 효율적인 주문 관리를 위한 애플리케이션입니다. 이 프로젝트는 학습 및 포트폴리오 목적으로 개발되었습니다.

## 🎯 주요 기능

- 주문 생성 및 관리
- 주문 상태 추적
- 고객 정보 관리
- 주문 내역 조회
